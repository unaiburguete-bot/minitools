Fórmula y metodología

Unidades de equilibrio = costes fijos ÷ (precio unitario − coste variable unitario).

Ejemplo práctico

Con 3.000 € de costes fijos y 30 € de margen de contribución necesitas vender 100 unidades.

La herramienta prioriza transparencia: muestra las variables que utiliza y evita convertir una estimación en una promesa. Para decisiones económicas, laborales o fiscales, contrasta el resultado con documentación y asesoramiento adecuados.

Preguntas frecuentes

¿Qué es el margen de contribución?

Es el precio menos el coste variable unitario disponible para cubrir costes fijos.

¿Qué ocurre si el precio no cubre el coste variable?

No existe punto de equilibrio positivo; cada venta aumenta la pérdida.

¿Puedo usar servicios?

Sí. Trata cada servicio o proyecto como una unidad comparable.
